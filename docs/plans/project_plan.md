## Development Workflow

### Phase 1: Setup

1. Create Supabase project
2. Set up Python environment
3. Install dependencies
4. Initialize database schema
5. Create demo corpus

### Phase 2: Ingestion

1. Implement document loader
2. Build chunking logic
3. Set up embedding pipeline
4. Test ingestion with sample docs

### Phase 3: Retrieval

1. Implement similarity search
2. Test different retrieval strategies
3. Evaluate retrieval quality
4. Optimize for accuracy

### Phase 4: Generation

1. Design prompt templates
2. Implement answer generation
3. Add citation logic
4. Test with sample questions

### Phase 5: Orchestration

1. Build LangGraph workflow
2. Implement multi-step reasoning
3. Add error handling
4. Process full questionnaires

### Phase 5.5: Domain Module

Model the core domain (questionnaires and answers) with persistence and comparison support.

### Phase 6: Refinement

1. Evaluate end-to-end performance
2. Improve prompts and retrieval
3. Add evaluation metrics
4. Document learnings
