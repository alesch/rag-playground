## Demo Corpus Structure

### Technical Infrastructure (finanso system)

- **Platform**: Java 9
- **Application Server**: Wildfly 17
- **Cloud Provider**: Microsoft Azure
- **Database**: PostgreSQL (managed Azure service)
- **Architecture**: Multi-tenant SaaS

### Compliance Certifications

- **SOC 2 Type II**: Trust Services Criteria compliance
- **ISO 27001**: Information Security Management System
- **Azure Compliance**: Inherited controls from Azure platform

### Documentation Categories

#### 1. Technical Infrastructure Documents
- System Architecture Overview
- Security Controls (authentication, encryption, access controls)
- Data Storage & Processing architecture
- Disaster Recovery & Business Continuity

#### 2. Compliance Certifications
- SOC 2 Type II Documentation
- ISO 27001 Certification details
- Azure Compliance Inheritance (shared responsibility model)

#### 3. Operational Procedures
- Incident Response Plan
- Change Management Process
- Vendor Management procedures
- Employee Security Training program

#### 4. Sample Customer Questionnaire
- Typical questions from financial institutions
- Coverage: security, availability, confidentiality, privacy
